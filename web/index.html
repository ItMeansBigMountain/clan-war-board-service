Wilderness fights

Clan battle board

Built for leaders who want clear terms before the fight and clear records afterward.

Privacy and performance

Public worlds, private player tracking, low-lag sync.

Fight worlds are public to bring activity back to the Wilderness. Individual member performance can stay private on the website, and the plugin sends batched war telemetry instead of one request per visible player.

Worlds are public

Open challenges and completed records can show the world so clans know where action is forming.

Members control public stats

Members can keep their player name off public performance pages while still helping clan-level war scoring.

Batched uploads

The plugin queues local fight events and flushes them in small batches to avoid war-time API spam and client lag.

Leader agreement

Terms every fight should lock before it starts.

Both leaders should confirm the same terms before members are rallied. Any change should require both clans to reconfirm.

Direct challenges

Leaders can call out a specific clan.

Challenge a named clan, send the proposed terms, accept counter-offers, then lock the match when both leaders agree to the same terms hash.

Clan directory

Only clans using the plugin show up here.

No random external clan extraction. A clan profile appears after its leaders/members use Clan War Board, then the in-depth view becomes fight history, member count, records, kills, deaths, returns, damage, and public stats.

Open fight board

Upcoming clan challenges.

Fight worlds are public to help clans find active Wilderness fights. Private leader notes stay with the participating clans until the fight is accepted.

Competitive table

Leaderboard built from plugin-confirmed wars.

Only clans that use Clan War Board appear here. Ratings stay unrated until completed plugin-tracked fights have enough confidence to count.

Winner system

How Clan War Board decides who won.

The winner system weighs kills, deaths, returns, location control, pressure, and third-party interference against the terms both leaders accepted.

War records

Completed battle results.

Published results focus on what matters: duration, participants, kills, returns, pressure, interference, and the agreed outcome.
